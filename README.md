# Portable Pinball

**Portable Pinball** is a compact 3D-printed mechanical toy designed using CAD software and fabricated with a 3D printer. The project focuses on creating a small, lightweight, and playable pinball machine that is easy to carry, assemble, and customize.

This project was designed as a hands-on maker project to demonstrate how CAD design, 3D printing, and basic mechanical mechanisms can be combined to create a fun and functional toy.

## Project Concept

Portable Pinball is designed to be a mini mechanical pinball machine that can be printed, assembled, and played without electronics. The main idea is to create a simple but fun game that uses mechanical movement such as a ball launcher, flippers, rails, and obstacle parts.

The playfield uses a **modular obstacle system**, allowing the obstacles to be adjusted, replaced, or rearranged. This makes the gameplay more flexible and different each time it is played.

## Key Features

* Compact and portable design
* Fully 3D-printed main structure
* Mechanical ball launcher
* Left and right flippers
* Modular obstacle system
* Adjustable and replaceable playfield parts
* Foldable stand
* Ball collection tray
* Easy to assemble and modify
* Educational project for CAD, 3D printing, and mechanical design

## Design and Fabrication

The project was designed using **Onshape** and produced using 3D printing technology. The design process included studying the pinball mechanism, creating the CAD model, preparing the files for 3D printing, printing the parts, assembling the mechanism, testing, and improving the design.

A custom spring mechanism was also used for the launcher and flipper action.

## Innovation

The main innovation of this project is the combination of **portable size**, **mechanical playability**, and **modular obstacle design**. Unlike a fixed pinball toy, this design allows users to change the obstacle layout and create different gameplay styles.

The project is also educational because it helps learners understand:

* CAD modeling
* 3D printing
* Mechanical movement
* Design testing and improvement
* Assembly and problem solving
